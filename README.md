# LoopForge

LoopForge is a playable AI game-design lab for Vibe Jam 2026: Hermes generates the mutations, you survive them.

## MVP loop

- Top-down arcade survival
- Collect cyan shards
- Avoid pink enemy loops
- Survive short waves
- Choose a mutation/upgrade between waves
- No login, no account, no multiplayer

## Local commands

```bash
npm install
npm run dev
npm test
npm run build
```

The Vibe Jam widget is included in `index.html`:

```html
<script async src="https://vibej.am/2026/widget.js"></script>
```

## Controls

Use WASD or arrow keys.
