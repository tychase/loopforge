# Ethical Engagement Rules

## Purpose

This project studies why games are engaging, but it should not create exploitative addiction systems.

The game should be replayable because it is fun, satisfying, and skill-building — not because it manipulates the player.

## Allowed Engagement Techniques

Use:

- clear goals
- satisfying controls
- fast feedback
- fair challenge
- visible progress
- short sessions
- meaningful choices
- skill mastery
- surprise modifiers
- replay variety
- cosmetic randomness
- non-monetary rewards

## Avoid

Do not use:

- real-money gambling mechanics
- loot boxes
- hidden odds
- fake scarcity
- manipulative streaks
- guilt-based retention
- dark-pattern timers
- pressure to keep playing
- monetized randomness
- endless grind with no purpose

## Psychological Researcher Agent Responsibility

The Psychological Researcher Agent must classify mechanics as:

1. Healthy engagement
2. Neutral engagement
3. Risky engagement
4. Exploitative / reject

The agent should explain why.

## Preferred Design Philosophy

Optimize for:

- “That felt good.”
- “I understand how to get better.”
- “I want to try one more run.”
- “That was fair.”
- “The game surprised me.”
- “My choices mattered.”

Do not optimize for:

- “I feel trapped.”
- “I have to keep playing.”
- “I might miss out.”
- “The reward is random and I need to chase it.”
- “I do not know why I am still playing.”
